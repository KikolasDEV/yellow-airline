# Yellow Airline Client

Frontend de Yellow Airline construido con React 19, Vite y TypeScript.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run preview
npm run test:unit
npm run test:e2e
npm run test
```

## Responsabilidades principales

- búsqueda de vuelos y calendario interactivo
- autenticación y experiencia VIP
- selección de pasajeros y asientos
- checkout con Stripe
- panel `My Bookings`
- i18n y tema claro/oscuro

## Testing

- `Vitest` para lógica y helpers
- `Playwright` para flujos E2E del navegador

Consulta la documentación general en el `README.md` raíz y `docs/testing.md`.
