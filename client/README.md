# Yellow Airline Client

Frontend de Yellow Airline construido con React 19, Vite y TypeScript.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run test:unit
npm run test:e2e
npm run test
```

## Testing

- `Vitest` para logica de negocio pura
- `Playwright` para smoke tests E2E del navegador

Consulta la documentacion general en el `README.md` raiz y `docs/testing.md`.
